Каталог географических точек и местоположений.

== Description ==

Сохраняйте географические координаты, делайте к ним описания, групперуйте их. Делитесь интересными местами на страницах своего сайта.

Основные возможности:

* Сохранение точек координат
* Произвольные маркеры для точек координат
* Группировка точек в слои
* Шорткоды для вставки карт и слоёв на страницы сайта

== Installation ==

1. Upload `wp-ds-geography.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
